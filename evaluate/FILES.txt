RAG + Classifier评测系统 - 文件清单
=====================================

创建时间: 2025-12-11
总文件数: 19个Python文件 + 5个JSONNET配置 + 3个文档 = 27个文件

## 主要文件

1. evaluate/config.yaml
   - 主配置文件
   - 包含分类器、LLM、检索器设置

2. evaluate/run_pipeline.py
   - 主流程调度器
   - 协调三个阶段的执行

3. evaluate/stage1_classify.py
   - 阶段1：分类实现
   - 使用Qwen 2.5-3B LoRA分类器

4. evaluate/stage2_generate.py
   - 阶段2：生成实现
   - 根据分类路由到不同的RAG流程

5. evaluate/stage3_evaluate.py
   - 阶段3：评估实现
   - 计算EM和F1指标

## 依赖项

核心依赖:
- unsloth (分类器)
- transformers, peft, torch (模型)
- vllm 或 fastapi (Llama服务)
- pyyaml (配置)
- tqdm (进度条)
- requests (HTTP)

## 使用流程

1. 启动Llama服务器
2. 启动检索服务
3. 运行: python evaluate/run_pipeline.py
4. 查看结果: evaluate/outputs/stage3_metrics/

## 预计运行时间

- 阶段1: 10-15分钟
- 阶段2: 4-8小时
- 阶段3: 1-2分钟
- 总计: 4-8小时（3000个样本）
